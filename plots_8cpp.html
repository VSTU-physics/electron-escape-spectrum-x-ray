void plot_k ()
 plot_k строит график интегральной функции выхода

\[ K(t) = \frac{\sum_i P_i \int_0^t q_i(z) dz} {\sum_i P_i \int_0^\infty q_i(z) dz}, \]

где \( P_i \) – вероятность рождения электрона \( i \)-ой группы, \( q_i(z) \) – плотность вероятности выхода этого электрона с глубины \( z \). Другими словами, это отношение числа электронов, вылетающих из образца с конечной толщиной к числу электронов, вылетающих из полубесконечного образца.

Подробное описание

В этом файле описаны функции для построения графиков.

Графики строятся при помощи программы Gnuplot. В Windows используется Wxt, в остальных ОС - qt терминал.

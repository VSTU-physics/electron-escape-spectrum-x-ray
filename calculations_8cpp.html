cubic_spline ищет коэффициенты сплайна:

\[ S_i(x) = a_i + b_i (x - x_i) + \frac{1}{2} c_i (x - x_i)^2 + \frac{1}{6} d_i (x - x_i)^3 \quad x\in[x_i, x_{i+1}]. \]

Аргументы
[in]xТочки \( x_i \), в которых нам известны значения функции \( y_i \)
[in]yМассив \( y_i \)
[in]NРазмер массивов
[out]aМассив коэффициентов \( a_i \)
[out]bМассив коэффициентов \( b_i \)
[out]cМассив коэффициентов \( c_i \)
[out]dМассив коэффициентов \( d_i \)

Решение задачи сводится к решению системы линейных уравнений с трёхдиагональной матрицей относительно вектора \( c_i \). Код:

1 double A[N-2], B[N-2], C[N-2], D[N-2];
2 double dx[N-1], dydx[N-1];
3 c[0] = 0;
4 c[N-1] = 0;
5 dx[0] = x[1] - x[0];
6 dydx[0] = (y[1] - y[0])/dx[0];
7 for (int i = 0; i<(N-2); i++)
8 {
9  dx[i+1] = x[i+2] - x[i+1];
10  dydx[i+1] = (y[i+2] - y[i+1])/dx[i+1];
11  A[i+1] = dx[i];
12  B[i] = 2*(dx[i] + dx[i+1]);
13  C[i] = dx[i];
14  D[i] = 6*(dydx[i+1] - dydx[i]);
15 }
16 for (int i = 1; i<N-2; i++)
17 {
18  B[i] -= A[i]/B[i-1]*C[i];
19  D[i] -= A[i]/B[i-1]*D[i-1];
20 }
21 c[N-2] = D[N-3]/B[N-3];
22 for (int i = N-4; i>=0; i--)
23 {
24  c[i+1] = (D[i] - C[i] * c[i+2])/B[i];
25 }

Остальные значения коэффициентов могут быть найдены как линейные комбинации \( c_i \), что реализует следующий код:

1 for (int i = 0; i<N-1; i++)
2 {
3  a[i] = y[i];
4  b[i] = dydx[i] - c[i]*dx[i]/3 - c[i+1]*dx[i]/6;
5  d[i] = (c[i+1] - c[i])/dx[i];
6 }

Обратите внимание 3 коэффициента остаются произвольными и их выбор связан с тем, как мы задаём приближение на концах. В данном случае они выбраны следующими:

1 d[N-1] = 0;
2 b[N-1] = 0;
3 a[N-1] = y[N-1];

Проверяем, что промежуток интегрирования внутри промежутка, на котором задана функция:

1 if ((la>=x[0])&&(la<=x[N-1])&&(lb>=x[0])&&(lb<=x[N-1]))

Интегрируем:

1 double a[N], b[N], c[N], d[N], h;
2 cubic_spline(x, y, N, a, b, c, d);
3 double result = 0.;
4 for (int j = 1; j<N-1; j++)
5 {
6  if (la<x[0])
7  {
8  h = x[0] - la;
9  result+=a[0]*h + b[0]*h*h/2 + c[0]*h*h*h/6 + d[0]*h*h*h*h/24;
10  }
11  if (lb>x[N-1])
12  {
13  h = lb - x[N-1];
14  result+=a[N-1]*h + b[N-1]*h*h/2 + c[N-1]*h*h*h/6 + d[N-1]*h*h*h*h/24;
15  }
16  if ((la<=x[j])&&(lb>=x[j+1]))
17  {
18  h = x[j+1] - x[j];
19  result+=a[j]*h + b[j]*h*h/2 + c[j]*h*h*h/6 + d[j]*h*h*h*h/24;
20  }
21  if ((la>x[j])&&(la<x[j+1]))
22  {
23  h = x[j+1] - la;
24  result+=a[j]*h + b[j]*h*h/2 + c[j]*h*h*h/6 + d[j]*h*h*h*h/24;
25  }
26  if ((lb>x[j])&&(lb<x[j+1]))
27  {
28  h = lb - x[j];
29  result+=a[j]*h + b[j]*h*h/2 + c[j]*h*h*h/6 + d[j]*h*h*h*h/24;
30  }
31 }

spe решает уравнение:

\[ \frac{\partial f}{\partial t} = p(t) \frac{\partial^2 f}{\partial z^2} + g(t), \]

при граничных условиях:

\[ a_1(t) f + b_1(t) \frac{\partial f}{\partial z} \Bigg|_{z = z_1} = c_1(t), \]

\[ a_2(t) f + b_2(t) \frac{\partial f}{\partial z} \Bigg|_{z = z_2} = c_2(t), \]

и произвольных начальных условиях неявным методом на одном шаге \([t, t+dt]\).

Аргументы
[out]fСодержит новый результат решения уравнения на одном шаге
[in]f_prevСодержит предыдущий результат решения уравнения на одном шаге
[in]zМассив узлов, над которыми решается уравнение
[in]NЧисло узлов
[in]dtШаг по времени (энергии)
[in]ptПараметр \(p(t)\)
[in]gtПараметр \(g(t)\)
[in]a_1tПараметр \(a_1(t)\)
[in]b_1tПараметр \(b_1(t)\)
[in]c_1tПараметр \(c_1(t)\)
[in]a_2tПараметр \(a_2(t)\)
[in]b_2tПараметр \(b_2(t)\)
[in]c_2tПараметр \(c_2(t)\)

Код:

1 A[0] = 0;
2 B[0] = a_1t - b_1t/(z[1] - z[0]);
3 C[0] = b_1t/(z[1] - z[0]);
4 D[0] = c_1t;
5 for (int i = 1; i < N - 1; i++)
6 {
7  double dz1, dz2, dza;
8  dz1 = z[i] - z[i-1];
9  dz2 = z[i+1] - z[i];
10  dza = (dz1 + dz2)*0.5;
11  A[i] = dt*pt/dz1/dza;
12  B[i] = - 1 - 2*pt/dz1/dz2*dt;
13  C[i] = dt*pt/dz2/dza;
14  D[i] = - gt*dt - f_prev[i];
15 }
16 A[N-1] = - b_2t/(z[N-1] - z[N-2]);
17 B[N-1] = a_2t + b_2t/(z[N-1] - z[N-2]);
18 C[N-1] = 0;
19 D[N-1] = c_2t;

задаёт трёхдиагональную матрицу и правую часть уравнения. A, B, C – диагонали матрицы нижняя, собственно диагональ, верхняя. D – правая часть уравнения. Код:

1 for (int i = 1; i<N; i++)
2 {
3  B[i] -= A[i]/B[i-1]*C[i-1];
4  D[i] -= A[i]/B[i-1]*D[i-1];
5 }
6 f[N-1] = D[N-1]/B[N-1];
7 for (int i = N-2; i>=0; i--)
8 {
9  //if (fabs(f_prev[i]) > 0) printf("%e %e\n", f[i], f_prev[i]);
10  f[i] = (D[i] - C[i] * f[i+1])/B[i];
11 }

Реализует метод прогонки решения систем линейных уравнений с трёхдиагональными матрицами.
